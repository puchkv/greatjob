ІНСТРУКЦІЯ КОРИСТУВАЧА
=================================================
1. Установіть Open Server версії 5.2.2
2. Запустіть сервер, після запуску на панелі завдань, в треє повинен з'явитися флажок красного кольору,
це свідчить про те, що сервер знаходиться у вимкнутому стані.
3. Натисність правою кнопкою миші на флажок, у випливаючому меню виберіть "Настройки".
4. Перейдіть на вкладку "Модули" та встановіть наступні значення:
	HTTP : Apache-PHP-7.2-x64 (або x86)
	PHP : PHP-7.2-x65 (або x86)
	Mysql/MariaDB : MySQL-5.7-x64
	PostgreSQL : Не использовать
5. Збережіть налаштування.
6. Перейдіть до папки, в яку ви встановили сервер. 
	Наприклад: С:\OpenServer\
7. Перейдіть до папки domains, та помістіть там папку "greatjob.ua", яка йде разом з цією інструкцією.
8. Тепер потрібно додати базу даних, для цього натисніть правою кнопкою миші на флажок на панелі завдань,
наведість курсор на опцію "Дополнительно" та оберіть "PhpMyAdmin".
9. Відкроється браузер з вікном авторизації, типово підходять наступні дані:
	login : root
	password:
10. Після того, як ви авторизуєтесь натисніть на вкладку "Импорт" зверху сторінки.
11. Натисніть кнопку "Обзор" та оберіть файл GreatJob.sql, що йде разом з цією інструкцією.
12. Натисніть кнопку "Вперед".
13. Натисніть на флажок на панелі завдань правою кнопкою миші та оберіть "Перезапустить".
14. Після короткого очікування флажок повинен набути зеленого кольору, це свідчить про те, що сервер наразі працює.
15. Якщо після перезапуску флажок залишився красного кольору, натисніть ще раз правою кнопкою миші на флажок та оберіть "Запустить"
16. Дивитися пункт 14.
17. Натисніть правою кнопкою миші на флажок, та наведіть курсор миші на опцію "Мои сайты",
та оберіть greatjob.ua
18. Повинен запуститися браузер з відкритим сайтом.
